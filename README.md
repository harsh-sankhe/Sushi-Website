# 🍣 Sushi Website

A responsive landing page for a sushi restaurant built using HTML, CSS, and Vite. This project focuses on creating a modern UI with responsive layouts and clean styling.

## 🚀 Tech Stack

- HTML5
- CSS3
- Vite

## ✨ Features

- Responsive design
- Modern landing page UI
- Smooth scrolling
- Modular CSS structure
- Clean and simple layout

## 📂 Project Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/harsh-sankhe/Sushi-Website/
cd SushiWebsite
npm install
```

Start the development server:

```bash
npm run dev
```
